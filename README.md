## Evaluations

We evaluate the agent using the following criteria:

* instructions_follow: The agent followed the user's instructions
* instructions_avoid: The agent avoided doing things it was told not to do
* answer_relevant: The response directly addresses the user's question
* answer_clear: The answer is clear and correct
* answer_citations: The response includes proper citations or sources when required
* completeness: The response is complete and covers all key aspects of the request
* tool_call_search: Is the search tool invoked?

We do this in two steps:

* Generate Synthetic Questions:
First, we use the Question Generation Agent to generate synthetic questions (refer to eval/data-gen.ipynb). The log files containing the generated questions are then saved in the eval/logs folder.
* Evaluate the Generated Questions:
Next, we run the Evaluation Agent on the generated questions to assess them based on predefined criteria (refer to eval/evaluations.ipynb).

Current evaluation metrics from evaluations notebook
```bash
instructions_follow    72.368421
instructions_avoid     92.105263
answer_relevant        51.315789

```
### Improvements

Currently, our evaluation is based on only 40 synthetic questions generated by the Question Generation Agent. To improve the robustness and reliability of our evaluation, we need to collect a larger, more diverse set of questions. Expanding the dataset will provide a more comprehensive evaluation and better insights into the performance of the generated questions.
